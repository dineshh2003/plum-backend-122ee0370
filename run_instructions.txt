NEXT STEPS:
1) Copy .env.example to .env and edit if needed:
   cp .env.example .env

2) To run locally (from project root):
   # activate (optional)
   source .venv/bin/activate
   uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

   then open: http://127.0.0.1:8000/docs

3) To run tests:
   source .venv/bin/activate
   pytest -q

4) To make server publicly reachable (ngrok):
   ngrok http 8000

5) To build docker:
   docker build -t backend-demo:latest .
   docker run -p 8000:8000 backend-demo:latest

6) GitHub:
   Create repo on GitHub, then:
   git remote add origin git@github.com:YOUR_USERNAME/$PROJECT_DIR.git
   git branch -M main
   git push -u origin main
